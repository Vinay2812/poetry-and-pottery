Customize Step 1: Category Selector

First step of the customize page - Category selection component with search and infinite scroll pagination. Mobile-first design supporting large numbers of categories.

Awaiting Approval

REQUIREMENTS

  • Support for MANY categories (scalable)
  • Search functionality to filter categories
  • Infinite scroll pagination
  • Mobile-first responsive design
  • Clear selected state
  • Show options count and price range per category

DESIGN GOALS

  • Fast search with real-time filtering
  • Lazy loading with skeleton states
  • Prominent selected state (green border)
  • Category image, name, options count
  • Base price range display
  • Sticky "Continue" button on mobile
  • Responsive grid layout

Notes / Feedback
Implementation Specs (Customize Step 1 - Category Selector):

• Search: Real-time filtering with debounce (300ms), clear button when text present
• Infinite Scroll: Intersection Observer to load more categories (12 per batch)
• Skeleton Loading: Show skeleton cards while loading more categories
• Selected State: Green border (#4F6F52) + subtle green background (#f8faf8), checkmark icon
• Category Card Data: Image, name, options count, base price range
• Mobile: 2-column grid, sticky "Continue" button at bottom
• Tablet: 3-column grid, inline continue button
• Desktop: 4-column grid, inline continue button
• Accessibility: Keyboard navigation, focus states, ARIA labels
• Files: src/features/customize/components/category-selector.tsx, category-card.tsx
