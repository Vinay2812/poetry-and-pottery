Snap Scrolling View All CTA Card Quick Add on Hover Arrow Nav + Counter Dot Indicators (Mobile) Fade Edges
Implementation Specs (Task 5.5):

• Layout: display: flex; overflow-x: auto replaces CSS grid
• Snap: scroll-snap-type: x mandatory with snap-start on cards
• Card Widths: 156px mobile, 200px tablet, 240px desktop (shows partial peek card)
• Scrollbar: Hidden via scrollbar-hide utility (Tailwind plugin)
• Touch: -webkit-overflow-scrolling: touch for iOS momentum
• End Card: "View All in [Category]" CTA card at end of scroll row, links to /shop?category=slug
• Quick Add (Desktop): Button slides up on card hover via CSS transform
• Arrows (Desktop): scrollBy({ left: cardWidth + gap, behavior: 'smooth' }) with card counter
• Fade Edge: CSS gradient pseudo-elements (40px), pointer-events: none
• Dots (Mobile): Dot pagination synced to scroll position via onScroll
• Accessibility: role="list" on container, aria-label for screen readers
