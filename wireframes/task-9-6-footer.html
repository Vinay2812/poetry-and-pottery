Task 9.6

Footer Design (Redesigned)

Site-wide footer redesigned to match the forest theme palette. Previous dark backgrounds (#1a1a1a, #2d3b2d) replaced with theme-appropriate colors. Three responsive options across desktop, tablet, and mobile.

Design Context

Before (Issues)

  • Dark backgrounds (#1a1a1a, #2d3b2d) don't match theme
  • Too heavy contrast against warm page backgrounds
  • Colors feel disconnected from forest/pottery palette
  • Harsh visual break between content and footer

After (Goals)

  • Use only approved theme colors (cream, primary-light, primary)
  • Smooth visual transition from page content to footer
  • Warm, cohesive feel matching pottery/nature brand
  • Responsive layouts for desktop, tablet, and mobile
Option A: Warm Cream Footer RECOMMENDED
Uses #F5F0E8 (cream) background — the warmest theme neutral. Creates a gentle, inviting base that feels like natural parchment. Green headings and primary CTA buttons add brand color without visual heaviness. Best match for the pottery/artisan aesthetic.

Implementation Notes

  • Container: Full-width background, content max-width 1200px centered
  • Padding: Desktop py-12 px-6, Tablet py-10 px-6, Mobile py-8 px-4
  • Grid: CSS Grid with grid-template-columns, responsive breakpoints at 768px and 640px
  • Typography: Headings use font-display (Plus Jakarta Sans), links use font-body (Outfit)
  • Theme Colors Only:
      Option A: bg-cream (#F5F0E8) — warmest, pottery feel
      Option B: bg-primary-light (#E8ECE8) — cool forest green tint
      Option C: bg-primary (#4F6F52) — bold brand green
  • Social Icons: 36px desktop, 32px mobile, rounded-full with subtle tinted background
  • Newsletter: Form uses flex layout, stacks vertically on mobile
  • Accordion (Mobile): Use Radix Accordion or custom state, animate height with data-[state=open]
  • Payment Icons: Consider using actual SVG brand icons or payment provider logos
  • Links: opacity-65 default, opacity-100 on hover with transition-opacity
  • Bottom Bar: Subtle top border using theme-tinted color, flex layout with justify-between
  • Accessibility: All links have descriptive text, proper heading hierarchy, keyboard navigable
