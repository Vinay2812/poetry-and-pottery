Task 4.2: Badge Component

Foundation component: Consistent badge/pill design with color variants for all statuses. Used for order status, event levels, stock status, and general labels.

Awaiting Approval

BEFORE (Current)

  • Inconsistent badge styles across components
  • Mixed border-radius values
  • Different color palettes per component
  • No standardized sizing

AFTER (New Design)

  • Unified badge component with variants
  • Consistent border-radius (pill, rounded, or soft)
  • Forest theme color palette throughout
  • Standardized sizes: sm, md, lg
  • Status variants for orders, events, products
