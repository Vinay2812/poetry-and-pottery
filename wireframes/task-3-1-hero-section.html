Task 3.1: Hero Section Redesign

Increase hero impact with taller aspect ratios, dual CTA buttons, larger heading text, and text reveal animation. Mobile: aspect-[4/5], Desktop: aspect-[2.5/1].

Status: Implemented
Full-Bleed Image with Bottom-Aligned Content

Full-bleed image hero with content bottom-left. 72px heading on desktop, softer gradient (70% opacity), shadow-xl. CTAs: h-11/h-12, rounded-[10px], 15px font. Badge: 11px, tracking-widest, uppercase. Container padding: py-4/py-6 for breathing room around rounded corners.

Mobile: aspect-[4/5] (portrait, tall and immersive). Badge + heading + subtitle + dual CTAs at bottom.
Desktop: aspect-[2.5/1] (wider cinematic). Large 72px heading, content bottom-left, image fills entire area.
CTAs: "Explore Collection" (white solid, rounded-lg) + "Learn More" (white outline, rounded-lg).
Animation: Text Generate Effect on heading (words reveal one by one with opacity).
Advantage: Preserves the existing visual design language. The pottery image stays as the dominant visual.

Task 3.1 Implementation Specs:

• Aspect Ratio: Mobile: aspect-4/5 (portrait). Desktop: aspect-[2.5/1] (cinematic).
• Heading Size: text-4xl (36px mobile) / lg:text-7xl (72px desktop). font-bold, tracking-tight. mb-5 lg:mb-6.
• Container Padding: py-4 lg:py-6 (16px mobile / 24px desktop breathing room for rounded corners).
• Content Padding: p-7 lg:p-16 (28px mobile / 64px desktop from edges).
• Badge: text-[11px], tracking-widest, uppercase, bg-white/15 backdrop-blur-md, rounded-full.
• Subtitle: text-sm lg:text-lg (14px/18px), font-normal, text-white/85, max-w-lg.
• Gradient Overlay: from-black/70 via-black/10 to-transparent (softer than initial).
• Shadow: shadow-xl (not shadow-2xl, follows flat design philosophy).
• Dual CTAs: h-11 lg:h-12, rounded-[10px], px-7, text-[15px]. Primary: bg-primary (green #4F6F52) + white text. Secondary: white outline (border-white/40).
• Text Animation: TextGenerateEffect (staggered word reveal with opacity + blur).
• Rounded Corners: rounded-[2rem] mobile / lg:rounded-[2.5rem] desktop.
