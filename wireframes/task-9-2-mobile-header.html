Implementation Specs (Task 9.2):

• Height: h-14 (56px) fixed height
• Position: fixed top-0 left-0 right-0 z-50
• Background: bg-white/80 backdrop-blur-xl (glassmorphism)
• Default state: Subtle border-b border-neutral-200/50
• Scrolled state: shadow-sm, remove border, stronger blur
• Hide on scroll: Hides when scrolling down past 150px, shows when scrolling up
• Animation: Slide up/down via Framer Motion (300ms easeInOut)
• Icon buttons: 40px touch targets, rounded-full, hover:bg-black/5
• Back arrow: Show on all sub-pages, navigates to parent
• Page title: font-display font-bold text-base, centered with flex:1
• Logo: 28px leaf icon (bg-primary rounded-lg) + "Poetry & Pottery" text
• Badge: 16px min-width, bg-primary, text-white, font-bold, ring-2 ring-white
• User avatar: 32px, bg-gradient primary, initials, ring on hover
• Signed out: User icon button (opens sign-in modal)
• File: src/features/layout/components/mobile-header.tsx
• Container: src/features/layout/containers/mobile-header-container.tsx
