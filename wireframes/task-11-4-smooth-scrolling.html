What Is Smooth Scrolling?

Smooth scrolling replaces the browser's default instant, stepped scroll behavior with a fluid, interpolated motion. Instead of content jumping abruptly when you scroll with a mouse wheel or trackpad, the page glides to its new position with easing — giving a premium, polished feel similar to high-end design websites and native apps.

🎨

Premium Feel

Smooth, interpolated scrolling adds a polished, luxury feel that matches the handcrafted pottery brand.

🔧

Scroll-Linked Animations

Enables precise scroll-triggered animations for product cards and section reveals (pairs with Task 11.3).

⚓

Anchor Scrolling

Smooth navigation to page sections (e.g. "Jump to Reviews", category anchors) instead of instant jumps.

📱

Touch-Aware

Configured to keep native touch behavior on mobile for performance while enhancing the desktop experience.

Behavior by Device

📱 Mobile NATIVE SCROLL

smoothTouch: false ensures mobile uses native touch scrolling for optimal performance and battery life.

Result: No smooth scroll effect on touch devices. Scroll feels native and responsive.
💻 Desktop (Trackpad) SMOOTH

Trackpad scrolling gets the full smooth interpolation effect. Two-finger scroll gestures feel buttery and refined.

Experience: Smooth, flowing scroll with momentum. Content glides instead of stepping.
🖱 Desktop (Mouse Wheel) SMOOTH

Mouse wheel notches are smoothed out. Instead of stepped jumps, each wheel tick produces a fluid scroll motion.

Experience: Most noticeable improvement. Wheel scrolling goes from choppy to silky.

Scope: Where Smooth Scroll Applies

Since Lenis wraps the root provider, smooth scrolling will apply globally to all public pages. The effect is most impactful on content-heavy scrollable pages.

● Products Pages
  • /products (shop listing)
  • /products/[id] (product detail)
  • Homepage product carousels
  • Related products section
● Events Pages
  • /events (all workshops)
  • /events/[id] (event detail)
  • /events/upcoming
  • /events/past
● Other Pages
  • Homepage (hero + sections)
  • Cart & Checkout
  • Orders & Wishlist
  • About / Contact
● Excluded
  • Dashboard (admin area)
  • Modals & Sheets (use data-lenis-prevent)
  • Dropdown menus

Important Considerations

⚠️

Modal & Sheet Scrolling

Modals, sheets, and dropdowns need data-lenis-prevent attribute to use native scrolling and avoid conflicts with the smooth scroll wrapper.

⚠️

Infinite Scroll Compatibility

The existing IntersectionObserver-based infinite scroll trigger works with Lenis since Lenis uses native scroll positions. No changes needed.

✓

Reduced Motion

Lenis automatically respects prefers-reduced-motion OS setting. When enabled, smooth scrolling is disabled and native behavior is used.

✓

Keyboard Navigation

Lenis preserves keyboard scrolling (arrow keys, Page Up/Down, Home/End, Space). Focus-based scrolling also works normally.
