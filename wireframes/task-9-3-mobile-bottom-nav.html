Task 9.3: Mobile Bottom Navigation

Fixed bottom navigation with frosted glass backdrop, animated active-state pill indicator, and badge counters with ring effect. 4 icon-only items: Home, Shop, Cart, Events. Light and dark theme support.

Awaiting Approval

Design Highlights

◼
Frosted Glass
Semi-transparent background with 24px backdrop blur. Content subtly peeks through for depth and context.
●
Animated Pill
Active tab has a rounded pill that slides between items using Framer Motion spring animation.
☾
Dark Mode
Full dark theme with adjusted glass opacity, pill color, icon tint, and badge ring color.

Light & Dark Themes

Light Mode
Glass: bg-white/80 + backdrop-blur-xl. Pill: bg-neutral-100, rounded-2xl.
Dark Mode
Glass: bg-black/80. Pill: bg-neutral-800. Active: #6b8f6e. Badge ring: dark bg.

Technical Specifications

Layout
  • Position: fixed bottom-0 right-0 left-0
  • Height: h-16 (64px) + safe area
  • Grid: grid-cols-4 equal columns
  • Z-index: z-50 with isolate
  • Padding: px-2 py-2
  • Visibility: lg:hidden (mobile only)
Frosted Glass
  • Light: bg-white/80 backdrop-blur-xl
  • Dark: bg-black/80 backdrop-blur-xl
  • Border: border-t border-white/20
  • Blur radius: 24px (Tailwind xl)
Icons
  • Size: h-5 w-5 (20px)
  • Default: stroke-2, color #737373
  • Active: stroke-[2.5], color #4f6f52
  • Dark active: #6b8f6e
  • Transition: duration-200
Active Pill
  • Shape: rounded-2xl (16px)
  • Light: bg-neutral-100
  • Dark: bg-neutral-800
  • Animation: Framer Motion layoutId
  • Spring: bounce 0.15, duration 0.5
Badge
  • Size: h-4 min-w-[16px]
  • Background: bg-primary (#4f6f52)
  • Text: text-white text-[10px] font-bold
  • Position: -top-1.5 -right-1.5
  • Ring: ring-1 ring-white / ring-black
  • Animate: scale 0→1 via AnimatePresence
Navigation Items
  • Home: / — Home icon
  • Shop: /products — Store icon
  • Cart: /cart — ShoppingCart + badge
  • Events: /events — CalendarDays + badge
  • Route: exact for /, prefix for others
