Task 9.7

WhatsApp Contact Button

Floating action button for quick WhatsApp contact. Positioned at bottom-right for easy access on both desktop and mobile devices.

Design Context

Before (Issues)

  • No quick contact option
  • Contact form only
  • Users must search for help
  • High friction for inquiries
  • No real-time support indication

After (Goals)

  • One-tap WhatsApp access
  • Visible but not intrusive
  • Shows availability status
  • Optional pre-chat message
  • Mobile-optimized positioning

Implementation Notes

  • Position: fixed positioning with z-index: 50, below modals but above page content
  • Desktop: bottom-8 right-8 (32px), button size w-16 h-16 (64px)
  • Mobile: bottom-6 right-4 (24px/16px), button size w-14 h-14 (56px)
  • With Bottom Nav: bottom-20 (80px) on mobile to clear navigation
  • WhatsApp Link: https://wa.me/91XXXXXXXXXX?text=Hello... with pre-filled message
  • Shadow: shadow-lg with colored shadow matching button (shadow-green-500/40)
  • Hover: hover:scale-105 transform with transition-transform duration-200
  • Tooltip: Appears on hover with opacity and visibility transition
  • Pulse Animation: CSS keyframes with animation: pulse 2s ease-out infinite
  • Popup: Use Radix Popover or custom state, animate with data-[state=open] transitions
  • Accessibility: aria-label="Contact us on WhatsApp", focusable, keyboard accessible
  • Analytics: Track clicks with event: whatsapp_fab_click
