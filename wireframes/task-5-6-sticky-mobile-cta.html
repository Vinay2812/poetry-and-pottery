Implementation Notes

Behavior Specifications

Visibility Trigger: Use IntersectionObserver to detect when main CTA button leaves viewport. Show sticky bar when not intersecting.
Animation: Slide up from bottom with opacity fade. Duration: 200ms ease-out. Use transform for performance.
Safe Area: Add padding-bottom for devices with home indicator (iPhone X+). Use env(safe-area-inset-bottom).
Z-Index: Should be above page content but below modals/sheets. Recommended: z-50
