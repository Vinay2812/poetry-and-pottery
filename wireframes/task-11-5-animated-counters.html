Notes / Feedback
Implementation Specs (Task 11.5):

• Library: Framer Motion useSpring / useInView (Option A), native JS (Option B), custom CSS (Option C)
• Trigger: IntersectionObserver / Framer Motion whileInView
• Once: Animation plays only once (not on scroll back)
• Font: font-display (Plus Jakarta Sans), font-extrabold
• Suffix: Support for "+" suffix (e.g., "500+")
• Separator: Thousand separator for numbers >= 1000 (Options A & B)
• Reduced Motion: Respects prefers-reduced-motion - shows final value instantly
• File: src/components/shared/animated-counter.tsx (new component)
