Customized Cart Item

Display customization options and price breakdown for custom products in cart. Extends the approved cart item card design (Option A - Compact Stacked) to show customization details.

Customization Data Structure 

interface ProductCustomizationData {
  options: CustomizationOptionSnapshot[];  // SIZE, COLOR, SHAPE, TEXT
  totalModifier: number;                   // Sum of all price modifiers
}

Example: Custom Mug
- Base: 850 | Size: Large (+150) | Color: Sage Green (+50) | Text: "Happy Birthday" (+100)
- Total: 850 + 300 = 1,150

Implementation Specs:

• Custom Badge: Green filled badge with "Custom" text to distinguish from regular products
• Chips: Small rounded pills (10px font, 12px border-radius) for customization options
• Color Chip: Include color dot (10px circle) before color name
• Price Breakdown: Show "Base + Custom" calculation clearly
• Data: Use ProductCustomizationData with options array and totalModifier
• Same Product Check: Products with different customizations = different cart items (use composite key)
• File: src/features/cart/components/CustomizedCartItem.tsx
