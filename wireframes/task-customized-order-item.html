Customized Order Item Display

Display customized products in order list and order detail views. Extends the approved Order Card Grid design (Option A) to show customization indicators and details.

Customization Data Structure 

interface ProductCustomizationData {
  options: CustomizationOptionSnapshot[];  // SIZE, COLOR, SHAPE, TEXT
  totalModifier: number;                   // Sum of all price modifiers
}

- Base: Rs 850 | Size: Large (+Rs 150) | Color: Sage Green (+Rs 50) | Text: "Happy Birthday" (+Rs 100)
- Total: Rs 850 + Rs 300 = Rs 1,150

Implementation Specs:

• Order List (thumbnail): 48x48px squares with customization indicator (badge, ring, icon, or dot)
• Order Detail (full card): Full customization options list with price modifiers
• Custom Badge: "C" or "Custom" badge in green (#4F6F52) to distinguish from regular products
• Chips: Small rounded pills (10px font, 12px border-radius) for customization options
• Color Chip: Include color dot (10px circle) before color name
• Price Breakdown: Show "Base + Custom" calculation clearly in order detail
• Review Button: Customized products are reviewable - show review CTA for delivered orders
• Data: Use ProductCustomizationData with options array and totalModifier
• Files: src/features/orders/components/OrderItemCard.tsx, OrderListItem.tsx
