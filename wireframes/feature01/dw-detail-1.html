Design Notes:
Route: /events/daily-workshops/[registrationId]
Layout: Mobile = single column stacked; Tablet = centered max-width 680px; Desktop = two-column with sticky sidebar
Status badges: Confirmed (green), Pending (amber), Cancelled (red) with colored dot indicators
Pricing: INR currency with tier breakdown, calculation shown per row, savings callout
Timeline: Vertical with colored dots — green check for completed, glowing for current, grey for pending, red X for cancelled
Actions: Cancel available for Pending/Approved states only; Download Receipt for paid/confirmed; Rebook for cancelled
Cards: rounded-2xl, soft shadows, no borders, flat design philosophy
Fonts: Plus Jakarta Sans for headings/labels, Outfit for body text
