My Registrations - 4-Section View

User's registrations split into 4 collapsible sections: Upcoming Daily Workshops, Completed Daily Workshops, Upcoming Events, and Completed Events. Each section has a colored left border accent and count badge.

Sections: 4 collapsible sections, all expanded by default
DW Cards: Show workshop config name, date/time chips, stats, price, actions

  • ✓ 2-column grid within each section
  • ✓ DW cards: stats + price inline, date chips wrap

  • ✓ Divider line between DW and Event sections
  • ✓ Collapse icons on section headers
Implementation Notes:
Route: /events/registrations (My Registrations tab active)
Data queries: userRegistrations(filter) with type filtering for DW vs Event
4 Sections: Upcoming DW, Completed DW, Upcoming Events, Completed Events
Collapsible: All sections expanded by default, click header to collapse
DW Card Data: Workshop config name, selected date/time chips, total hours, participants, pieces, price
Status Badges: Pending (amber), Approved (green), Paid (blue), Confirmed (dark green), Completed (gray)
Cancel Button: Only visible for Pending and Approved statuses
Reviews: Completed registrations show "Write a Review" button or star rating
Tab Badge: Shows total count of active registrations (upcoming DW + upcoming events)
Section Accents: Green (upcoming active), Muted green (completed DW), Gray (completed events)
