Merged Events View

Combined "Upcoming" and "Past" events under the single "Events" tab. Replaces the previous tabbed upcoming/past separation with a continuous scrollable view. Users see everything at a glance with clear section headings and count badges.

Route: /events/events (Events tab active)
Components: EventsMergedContainer → EventsMergedView
Data: upcomingEvents + pastEvents queries merged into one page
Behavior: Single scroll view, "Show More" loads next batch per section
Filters: Event Type (All / Workshop / Open Mic / Masterclass), Sort (Soonest First / Latest First / Price Low-High)

Past events: Reduced opacity (0.75), grayscale(30%) on thumbnails
