Notes / Feedback
Implementation Notes:

3 Event States per Option:
- Upcoming: Register CTA, pricing, spots available, full event info
- Registered: Registration status card (Confirmed/Pending), ticket with QR code, Cancel Registration option
- Completed: Gallery with thumbnails, rating summary, reviews section, Write Review CTA

3 Layout Options:
- Option A (Stacked): Full-width hero on top, content + sidebar below. Classic and clean
- Option B (Side-by-Side): Image left, content right (50/50). Gallery-friendly, visual
- Option C (Magazine): Full-bleed hero with floating card overlay. Premium editorial feel

Additional States: Sold Out (disabled CTA) and Cancelled (cancellation notice + redirect) share the selected layout

Mobile CTA Bar: Sticky bottom bar on all mobile views — adapts per state (Register / View Ticket + Cancel / Write Review)

Registration Status: Shows Confirmed (green) or Pending (yellow) with registration ID, date, and amount
