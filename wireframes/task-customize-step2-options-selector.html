Customize Step 2: Options Selector

Second step of the customize page. After selecting a category, users choose customization options grouped by type (Size, Color, Shape, Text) with real-time price updates.

Awaiting Approval

CONTEXT FROM STEP 1

  • User selected a category (e.g., "Custom Mugs")
  • Category has base price (e.g., Rs 850)
  • Category defines available option types
  • Ready to customize with options

STEP 2 REQUIREMENTS

  • Show selected category with back/change option
  • Group options by type (Size, Color, Shape, Text)
  • Each option shows price modifier (+Rs 100)
  • Running total updates in real-time
  • Required options must be selected to continue
  • Continue/Add to Cart when complete

Notes / Feedback
Implementation Specs (Customize Step 2):

• Selected Category Header: Shows category image, name, base price with back/change option
• Option Types: SIZE (chip buttons), COLOR (circular swatches), SHAPE (image chips), TEXT (input field)
• Price Modifiers: Each option shows +Rs X in muted green text
• Running Total: Updates in real-time as selections change
• Required Options: Amber "Required" badge, Continue disabled until selected
• Mobile: Sticky footer with total and Continue/Add to Cart button
• Desktop: Summary sidebar with all selections and pricing breakdown
• Step Indicator: Shows progress through Category > Options > Review flow
• Files: src/features/customize/components/options-selector.tsx, option-type-accordion.tsx, size-chips.tsx, color-swatches.tsx
