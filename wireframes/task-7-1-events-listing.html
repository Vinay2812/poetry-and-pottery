Task 7.1: Events Listing Pages

Unified layout system for all event listing pages: Main events hub, Upcoming events, Past workshops, and User registrations. All pages share the same header pattern, tabs navigation, grid layout, and card spacing.

⏳ AWAITING APPROVAL

CURRENT BEHAVIOR

  • Inconsistent page headers across event pages
  • Different tab navigation patterns
  • Varying grid layouts and spacing
  • No unified card spacing system
  • Different responsive breakpoints

NEW DESIGN

  • Unified page header (title + underline + subtitle)
  • Consistent tabs with underline active state
  • Mobile: 1-col, Tablet: 2-col, Desktop: 3-col
  • Same card spacing: gap-4 (mobile), gap-6 (tablet), gap-8 (desktop)
  • Shared responsive behavior across all pages
