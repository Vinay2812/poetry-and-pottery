Implementation Notes:

Part 1 - Products Page:
- Page-level underline tabs (Active/Archive) using approved Task 4.9 Option A pattern
- Archive tab shows sold-out and discontinued products with badges
- Collections filter highlighted in sidebar with green background (#f8faf8) and left border
- Archive products have muted appearance (opacity: 0.6) and grayed-out prices

Part 2 - Homepage Carousel:
- Uses Embla Carousel with dragFree: true, align: "start", containScroll: "trimSnaps"
- No visible arrows - users drag/swipe to navigate
- Responsive sizing: 80% mobile, 34% tablet, 25% desktop
- Header pattern: Title + subtitle on left, "View All" link on right (tablet/desktop)
- Matches existing ProductCarousel component pattern exactly

Collection Card:
- 16:9 aspect ratio image with gradient placeholders
- Badges: "New" (green), "Limited" (amber) positioned top-left
- Item count badge positioned bottom-right
- hover: translateY(-4px) + enhanced shadow
