• Option A (Single Page) recommended for simplicity and fast completion

• Option D (Tabs) works well when preview needs to stay visible

Implementation Notes:
• Live preview should update dynamically as user selects options
• WhatsApp button should pre-fill message with selections summary
• Form should validate that at least size, color, finish, and shape are selected
• Custom text is optional (max 50 characters)
