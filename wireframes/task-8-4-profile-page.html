Implementation Notes

Design Specifications

  • Avatar: Initials on gradient background (#4F6F52 to #6B8F6E)
  • Card Corners: rounded-2xl (16px) for main cards, rounded-xl for inner
  • Shadows: shadow-soft default, shadow-card on hover
  • Quick Link Icons: 44-60px depending on viewport

Quick Links

  • Orders: Links to /orders - show count of total orders
  • Wishlist: Links to /wishlist - show count of saved items
  • Addresses: Links to /addresses - show count of saved addresses
  • Registrations: Links to /registrations - show count of upcoming events

Interactive Elements

  • Edit Profile: Opens profile edit modal/page
  • Quick Links: Hover shows shadow-card, click navigates
  • Sign Out: Triggers Clerk sign out, redirects to home
