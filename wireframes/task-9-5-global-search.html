Task 9.5: Global Search

Command palette style search dialog with grouped results (Products, Events, Orders), keyboard navigation, and full-screen mobile overlay. Covers all UI states: idle, loading, results, partial results, empty, and error.

Approved

CURRENT STATE

  • Tab-based category switching with icon buttons
  • No idle state (blank before typing)
  • No loading skeleton
  • Basic empty state per tab only
  • No partial results handling
  • No keyboard navigation
  • Same layout on mobile and desktop
  • No recent searches or suggestions

REDESIGNED

  • Command palette with grouped results
  • Idle state: recent searches + trending queries
  • Loading state with skeleton shimmer
  • Full results with counts and "View all" links
  • Partial results: shows available, hides empty
  • Rich empty state with search suggestions
  • Error state with retry action
  • Full-screen mobile overlay
  • Arrow key + Enter keyboard navigation
  • Cmd/Ctrl+K shortcut trigger

Interaction & Behavior Specs
Behavior Specification
Trigger Search icon click, or ⌘K / Ctrl+K
Dismiss ESC, backdrop click, or mobile back button
Debounce 300ms debounce on search input before API call
Min Query Search triggers after 1+ characters typed
Keyboard Nav ↑↓ to move selection, ↵ to navigate, Tab to switch tabs (Option B)
Max Results 3 items per group in Option A/C, 5 items in Option B tab view
View All Navigates to /products?search=..., /events?search=..., or /orders?search=...
Recent Searches Stored in localStorage, max 5 items, individually removable
Auto-focus Input focused immediately on open, body scroll locked
Animation Backdrop: fade 200ms. Dialog: fade + slide-down 200ms. Mobile: slide-right 250ms
Orders Tab Only shown for authenticated users

Implementation Specs (Task 9.5) — Approved: Option B (Tab Navigation)

• File: src/features/global-search/components/global-search.tsx
• Container: src/features/global-search/containers/global-search-container.tsx
• Hook: src/features/global-search/hooks/use-global-search.ts
• Dialog: max-w-[600px], rounded-2xl (20px), shadow-2xl, centered with top offset (lg:top-24)
• Backdrop: bg-black/50, backdrop-blur-sm
• Input: text-lg, no border, auto-focus, clear button on right
• Result Layout: Underline tabs (Products, Events, Orders) with count badges. Tab key to switch. 5 items per tab
• Orders Tab: Only shown for authenticated users
• Thumbnails: 48px, rounded-[10px]. Events use date block (month + day)
• Skeleton: Shimmer animation, 3 items
• Idle State: Recent searches (localStorage, max 5, clock icon, removable) + trending tags
• Mobile: Full-screen overlay, back button, underline tabs matching desktop. No footer shortcuts
• Keyboard: ⌘K/Ctrl+K open, ↑↓ navigate, ↵ select, Tab to switch tabs, ESC close
