Notes / Feedback
Implementation Specs (Task 13.1):

• Base color: bg-muted (#e7e5e4 / neutral-200)
• Animation: Shimmer gradient sweep (1.5s ease-in-out infinite)
• Corners: Match component (rounded-xl for cards, rounded-2xl for images)
• Reduced motion: Fallback to static background with prefers-reduced-motion
• Performance: Use transform and opacity only, no layout properties
• Files: src/components/ui/skeleton.tsx, src/components/skeletons/*

Accessibility:
• Use aria-busy="true" on skeleton containers
• Use role="status" for screen reader announcement
• Respect prefers-reduced-motion: reduce animations for users with motion sensitivity
