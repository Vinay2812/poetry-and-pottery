Customize Page

Customization page for pottery products. Users select a category and customize options (Size, Color, Shape, Text) with real-time price updates.

Awaiting Approval

CURRENT BEHAVIOR

  • No customization feature exists
  • Users cannot personalize products
  • No price modifier system
  • No visual feedback for selections
  • Limited product variety

NEW DESIGN

  • Category selection (Mugs, Vases, Bowls, etc.)
  • Options grouped by type (Size, Color, Shape, Text)
  • Real-time price calculation with modifiers
  • Visual feedback for selected options
  • Clear price breakdown in summary
  • Mobile-optimized step-by-step flow
  • Desktop sidebar with live preview

Notes / Feedback
Implementation Specs (Customize Page):

• Page Header: Consistent with other pages - breadcrumbs, title, underline accent
• Category Selection: Card or tab-based selection for Mugs, Vases, Bowls, Plates, etc.
• Option Types: SIZE (chips), COLOR (swatches), SHAPE (chips/cards), TEXT (input)
• Price Modifiers: Displayed as "+₹X" badges next to options
• Price Summary: Real-time calculation showing base + modifiers = total
• Mobile: Step wizard (A) or collapsible sections (B) with sticky footer CTA
• Desktop: Two-column layout with sticky summary sidebar
• Database: CustomizationOption table with category, type, name, value, price_modifier fields
• Files: src/features/customize/, src/app/customize/page.tsx
